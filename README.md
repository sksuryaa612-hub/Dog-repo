# 🐶 Task 16.1 - Random Dog Generator

## 📖 Project Description

The Random Dog Generator is a responsive web application that fetches and displays random dog images using the Dog API. The application is developed using HTML, CSS, and JavaScript without any external frameworks or libraries.

🐕 Users can view a random dog image when the page loads and generate a new image by clicking the button. The application communicates with the API to fetch real-time data and update the webpage dynamically.

⚡ This project demonstrates API integration, asynchronous programming using Async/Await, error handling with Try/Catch, and responsive web design principles.

🎨 The user interface is clean, modern, and mobile-friendly. A loading indicator is displayed while data is being fetched to provide a better user experience.

🛠️ The project follows professional coding standards by using semantic HTML, external CSS and JavaScript files, reusable functions, descriptive variable names, and a well-organized folder structure.
